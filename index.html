JavaScript Overview

JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and much more.

Here are some key features of JavaScript:

  • Lightweight and interpreted
  • Object-oriented
  • Imperative and functional
  • First-class functions
  • Client-side execution

JavaScript Syntax

JavaScript syntax is the set of rules that define how JavaScript programs are written. It borrows heavily from C and Java syntax.

Here are some basic syntax elements:

  • Variables: Declared using var, let, or const.
  • Data Types: Numbers, strings, booleans, objects, etc.
  • Operators: Arithmetic, comparison, logical, etc.
  • Control Flow: if-else statements, loops, etc.
  • Functions: Defined using the function keyword.

JavaScript Variables

Variables are used to store data values. JavaScript uses the keywords var, let, and const to declare variables.

Examples:


          var name = "John Doe";
          let age = 30;
          const PI = 3.14159;

JavaScript Operators

Operators are used to perform operations on variables and values.

Types of operators:

  • Arithmetic Operators: +, -, *, /, %, ++, --
  • Assignment Operators: =, +=, -=, *=, /=, %=
  • Comparison Operators: ==, ===, !=, !==, >, <, >=, <=
  • Logical Operators: &&, ||, !

JavaScript Control Flow

Control flow statements determine the order in which code is executed.

Examples:

  • if-else statements: Execute different blocks of code based on a condition.
  • Loops: Repeat a block of code multiple times.
    • for loop: Iterate a specific number of times.
    • while loop: Iterate as long as a condition is true.
  • switch statement: Execute different blocks of code based on different cases.

JavaScript Functions

Functions are reusable blocks of code that perform a specific task.

Example:


          function greet(name) {
            console.log("Hello, " + name + "!");
          }

JavaScript Objects

Objects are collections of key-value pairs. They are used to represent real-world entities.

Example:


          const person = {
            firstName: "John",
            lastName: "Doe",
            age: 30
          };

JavaScript Arrays

Arrays are used to store multiple values in a single variable.

Example:


          const colors = ["red", "green", "blue"];

JavaScript Strings

Strings are used to represent text.

Example:


          let message = "Hello, world!";

JavaScript DOM

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It allows you to manipulate the content, structure, and style of a website.

Example:


          // Change the text content of an element with id="myHeader"
          document.getElementById("myHeader").innerHTML = "New Header Text";

JavaScript Events

Events are actions or occurrences that happen in the browser, such as clicking a button, loading a page, or hovering over an element. JavaScript allows you to respond to these events.

Example:


          // Display an alert when a button is clicked
          <button onclick="alert('Button clicked!')">Click Me</button>
